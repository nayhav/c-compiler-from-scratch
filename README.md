## About

This is my attempt at writing a self-hosted C compiler that parses a subset of the C language, built incrementally to better understand how compilers work. Will continue building on this.

### Resources

This project is inspired by

- **Nora Sandler's _Writing a C Compiler_** – https://norasandler.com/2017/11/29/Write-a-Compiler.html  
  A fantastic step-by-step guide to building a C compiler, though it can be a bit challenging for beginners.

- **A Compiler Writing Journey (ACWJ)** – https://github.com/DoctorWkt/acwj  
  An excellent GitHub repository that provided inspiration and additional implementation ideas.

- **The SubC Compiler** - https://www.t3x.org/subc/
  Public domain compiler for a clean subset of C.

- Countless conversations with ChatGPT to help me understand compiler concepts and bridge gaps in my knowledge.

I plan to build this compiler incrementally, documenting each stage as I learn and understand the underlying concepts.

This project is being developed on **Windows 11** using **MSYS2 UCRT64** to provide a Unix-like development environment. Although the tutorial uses Linux, the implementation and build process have been adapted to work on Windows.
